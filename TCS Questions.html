👤 Self Introduction & Resume

💻 Coding Questions (Write Code)

🔢 Number-Based Programs

  • Fibonacci series Very Frequent
  • Factorial of a number Very Frequent
  • Check if a number is Prime Very Frequent
  • Reverse of a number
  • Palindrome check (number)
  • Armstrong number
  • Strong number
  • Write a program for recursion (e.g., calculate batsman strike rate and bowler's net run rate)

🔤 String-Based Programs

  • Reverse a string Very Frequent
  • Palindrome check (string)
  • Count each character in a string — e.g., "aaabbaccc" → a:4, b:2, c:3 Very Frequent
  • Delete / remove duplicates in a string
  • Reverse the words inside a sentence — e.g., "my name is kovidh" → "ym eman si hdivok"

📐 Pattern & Matrix Programs

  • Triangular star/number pattern
  • Matrix multiplication

📊 Complexity Analysis

  • Explain space complexity and time complexity of your code (e.g., character count program) Asked

🧱 OOPs Concepts

  • What is Inheritance? Very Frequent
  • What is Encapsulation? Very Frequent
  • What is Polymorphism? Very Frequent
  • What is Abstraction?
  • Difference between Method Overloading and Method Overriding Very Frequent
  • Write code demonstrating Method Overriding
  • Procedural Programming vs OOPs — differences Asked
  • Explain polymorphism with a real-world example
  • Explain encapsulation with a real-world example
  • Difference between .equals() and ==

🗄️ SQL & DBMS

  • What is DBMS?
  • What is SQL? Do you know SQL?
  • Write a SQL query using aggregate functions (SUM, COUNT, AVG, etc.) Frequent
  • What is GROUP BY clause? Write a query using GROUP BY
  • What are Constraints in DBMS? (NOT NULL, UNIQUE, PRIMARY KEY, etc.)
  • How to find the last inserted row in SQL?
  • How to add a column to an existing table? (ALTER TABLE)
  • Difference between MongoDB and MySQL
  • Write a code/query for MongoDB

🧠 MR Round — Managerial & Logical Questions

Logical / Aptitude Puzzles

  • Lotus pond puzzle: "In a pond, I pick one lotus, and for each lotus it doubles. In 30 days it covers half the pond. In how many days will it cover the entire pond?" (Answer: 31 days)
  • Today is Thursday. What is the day after 45 days?

Situational / Behavioral (MR)

  • If your teammate in office does not agree with your ideas, what will you do?
  • You and another member have conflicting approaches — how do you resolve it?
  • If two projects are assigned to you simultaneously, how will you manage?
  • If you have 2 tasks with the same deadline, how will you complete them?
  • If your team members can't work due to some reasons and you're the only one left, what will you do?
  • Will you be able to handle multitasking?
  • If assigned to a project using technologies you don't know, what will you do?

💡 Tips & Advice from Candidates

✅ Be Confident

Even if you don't know something, be confident and answer in a general way without fear. Be honest — interviewers appreciate honesty over bluffing.

✅ Project Explanation

Project explanation should be good but be specific and to the point. Don't go out of the topic. Mention problem statement → approach → tools used → outcome.

✅ Be Technical

Be a more technical person. If you mention any technology/certification on your resume, prepare in-depth questions on it. They WILL ask.

✅ Cloud Technologies

If you've added any cloud tech to your resume, go through the basics thoroughly (EC2, S3, IAM, etc.).

✅ Know Complexity

After writing code, be ready to explain its time complexity and space complexity.

✅ OSI Model

Prepare the OSI Model layers especially if you have a networking / cybersecurity background.

✅ Real-World Examples

For OOP concepts and ML algorithms, giving real-world examples adds great value to your answer.

✅ Duration

TR+MR round is typically 20-25 minutes. HR round is typically 15-20 minutes. Be concise and clear.
