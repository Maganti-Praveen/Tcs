6. MR — Puzzles & Aptitude ▼
🧩 Lotus Pond Puzzle — doubles every day, half in 30 days, full? Classic ▼
Answer: 31 days

Since the lotus doubles every day, the quantity on the next day becomes twice the previous day. If the pond is half covered on the 30th day, then on the next day it will double and cover the entire pond. So, the pond will be fully covered on the 31st day.

How to Explain

Don't rush to answer. Say the doubling logic clearly. Show your thought process.

🧩 Today is Thursday. What is the day after 45 days? Common ▼
Answer: Sunday

Days repeat every 7 days, so divide: 45 ÷ 7 = 6 weeks + 3 days remainder. Add 3 days to Thursday: Friday (1) → Saturday (2) → Sunday (3).

45 ÷ 7 = 6 remainder 3 Thursday + 1 = Friday Thursday + 2 = Saturday Thursday + 3 = Sunday ✅
In Interview

Speak your steps aloud. Even if wrong, logic matters more than the final answer.

🧩 Why are manhole covers round? Sometimes ▼
Answer

A round manhole cover cannot fall through the hole because the diameter is the same in all directions. A square or rectangular cover could fall through diagonally. Also, round covers are easier to roll and move.

🧩 You have 2 eggs and a 100-floor building. Find the critical floor with minimum attempts. Advanced ▼
Answer: 14 attempts

Use the interval approach: Drop the first egg from floors 14, 27, 39, 50, 60, 69, 77, 84, 90, 95, 99, 100. If the first egg breaks at floor N, use the second egg to check linearly from the previous checkpoint. Maximum attempts = 14.

Tip

If you don't remember exact solution, explain the approach: "I'd use decreasing intervals with first egg, then linear search with second egg."

What are the 4 pillars of OOP? Very Common ▼
Answer

1. Encapsulation — Wrapping data and methods together, restricting direct access.

2. Abstraction — Hiding complex implementation, showing only essential features.

3. Inheritance — One class inherits properties of another class.

4. Polymorphism — Same method behaves differently based on context (overloading/overriding).

Difference between SQL and NoSQL? Common ▼
Answer

SQL — Structured, relational, uses tables with rows/columns, fixed schema (MySQL, PostgreSQL).

NoSQL — Unstructured/semi-structured, flexible schema, document/key-value based (MongoDB).

SQL is better for complex queries and transactions. NoSQL is better for scalability and flexibility.

What is an API / REST API? Common ▼
Answer

API (Application Programming Interface) — A set of rules that allows two applications to communicate with each other.

REST API — An API that follows REST principles: uses HTTP methods (GET, POST, PUT, DELETE), is stateless, and communicates using JSON.

In my TrackMate project, I created REST APIs for user authentication, vehicle data, and location updates.

What is Cloud Computing? Common ▼
Answer

Cloud computing is the delivery of computing services — like servers, storage, databases, and software — over the internet. Instead of owning physical hardware, you use services from providers like AWS, Azure, or Google Cloud on a pay-as-you-use basis. Examples: ServiceNow runs on cloud. Netflix uses AWS.

What is Agile / Scrum? Sometimes ▼
Answer

Agile — A project management methodology that delivers work in small, iterative increments called sprints, with continuous feedback.

Scrum — A framework within Agile: team works in 2–4 week sprints, with daily standups, sprint planning, and retrospectives.

TCS uses Agile methodology in most projects.

What is Logistic Regression? (From your internship) Expected ▼
Answer

Logistic Regression is a classification algorithm used in machine learning to predict binary outcomes (Yes/No, Positive/Negative). In my AI/ML internship, I used it for sentiment analysis on customer reviews — classifying reviews as positive or negative. It uses a sigmoid function to map predictions to probabilities between 0 and 1.

What is Socket.IO? (From your project) Expected ▼
Answer

Socket.IO is a JavaScript library that enables real-time, bidirectional communication between the client and server. Unlike regular HTTP requests (request-response), Socket.IO keeps a persistent connection open so both sides can send data instantly. In TrackMate, I used it to send live vehicle location updates to users in real time without page refresh.

Difference between Java and JavaScript? Sometimes ▼
Answer

Java — Object-oriented, compiled, strongly typed, runs on JVM. Used for backend, Android, enterprise apps.

JavaScript — Scripting language, interpreted, dynamically typed, runs in browser and Node.js. Used for web frontend and backend.

They are completely different languages despite similar names.
